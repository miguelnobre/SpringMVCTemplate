# Spring-MVC Template

#### Add-ons:
- Log4J Configuration
- Spring-Security `4.2.3.RELEASE` (w/ Basic Auth)
- Embedded Tomcat 7 (`mvn tomcat7:run`)

#### Launch Server:
`sh run-server.sh` (Maven need to be configured)

#### How to access:
http://localhost:8080/spring-mvc-template/

#### Basic Auth Credentials:
User: `user` 
<br>
Password: `password`
